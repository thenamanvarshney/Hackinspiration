ExpenseAI: Project Requirements & Setup
1. System Requirements
To run this project, ensure you have the following installed:
Node.js: v16.0.0 or higher
NPM: v8.0.0 or higher (Usually comes with Node.js)
Web Browser: Chrome, Edge, or Brave (with Wallet support)

2. Key Libraries & Tech Stack
This project is built using the following core technologies. Running npm install will automatically download them.

Frontend Framework
React (v18.2): Library for building the user interface.
Vite (v4.4): Fast build tool and development server.
TypeScript (v5.0): For type-safe code and error prevention.
UI & Styling
Tailwind CSS (v3.3): Utility-first CSS framework for styling.
Notistack (v3.0): For popup notifications and alerts.
Lucide React: For scalable vector icons (Wallet, Trash, Edit icons).
Blockchain & Web3
algosdk (v2.4): The official Algorand JavaScript SDK for transaction construction.
@txnlab/use-wallet-react (v3.0): Hooks for connecting Pera, Defly, and KMD wallets.

3. How to Install & Run
Open Terminal inside the project folder: cd projects/frontend
Install All Libraries: npm install
Start the App: npm run dev
Open Browser: Go to http://localhost:5173

4. Troubleshooting
Node Modules Missing?: If you see "module not found" errors, delete the node_modules folder and run npm install again.
Wallet Connection Issues: Ensure your LocalNet is running or switch your wallet to Testnet mode.
